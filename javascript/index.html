¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, orientado a objetos, utilizado principalmente para crear páginas web interactivas. Es uno de los tres pilares del desarrollo web junto con HTML y CSS.

Paradigma

JavaScript es multiparadigma: soporta programación orientada a objetos, funcional y basada en eventos.

Tipos de variables

  • let: Permite declarar variables con alcance de bloque.
  • const: Permite declarar constantes, su valor no puede cambiar.

Tipos de datos

  • String: Cadena de texto
  • Number: Número
  • Boolean: Verdadero o falso
  • Object: Objeto
  • Array: Lista ordenada
  • Null: Ausencia de valor
  • Undefined: Valor no definido

Declaración de variables

let nombre = "Juan";
const PI = 3.14;
let edad = 20;

Objetos

let persona = {
  nombre: "Ana",
  edad: 25,
  esEstudiante: true
};

Arrays

let numeros = [1, 2, 3, 4, 5];

Funciones

function saludar() {
  console.log("Hola!");
}

Operadores aritméticos

Los operadores aritméticos permiten realizar operaciones matemáticas básicas:

  • + (Suma): let suma = 2 + 3; // 5
  • - (Resta): let resta = 5 - 2; // 3
  • * (Multiplicación): let producto = 4 * 2; // 8
  • / (División): let division = 10 / 2; // 5
  • % (Módulo): let resto = 7 % 3; // 1
  • ** (Exponenciación): let potencia = 2 ** 3; // 8

Operadores de comparación

Los operadores de comparación permiten comparar valores y devuelven un valor booleano (true o false):

  • == (Igualdad): 5 == "5" // true
  • === (Igualdad estricta): 5 === "5" // false
  • != (Desigualdad): 3 != 4 // true
  • !== (Desigualdad estricta): 3 !== "3" // true
  • > (Mayor que): 7 > 5 // true
  • < (Menor que): 2 < 4 // true
  • >= (Mayor o igual que): 6 >= 6 // true
  • <= (Menor o igual que): 3 <= 5 // true

Operadores lógicos

Los operadores lógicos permiten combinar o invertir condiciones:

  • && (AND): true && false // false
  • || (OR): true || false // true
  • ! (NOT): !true // false

Ejemplos de uso en variables

let edad = 20;
let esEstudiante = true;
let puedeEntrar = edad >= 18 && esEstudiante; // true
let acceso = edad < 18 || esEstudiante; // true
let noEstudiante = !esEstudiante; // false

Operador ternario

El operador ternario es una forma corta de escribir una condición. Su sintaxis es condición ? valorSiVerdadero : valorSiFalso:

let edad = 20;
    let mensaje = edad >= 18 ? "Mayor de edad" : "Menor de edad";
    console.log(mensaje); // "Mayor de edad"

    function obtenerEstado(usuario) {
      return usuario.activo ? "Activo" : "Inactivo";
    }

    // También se puede usar para asignar valores o ejecutar funciones:
    let resultado = esEstudiante ? aprobar() : reprobar();

El operador ternario es útil para simplificar condicionales y hacer el código más legible cuando solo hay dos opciones posibles.

Ejemplos de uso en funciones

function puedeComprarAlcohol(edad) {
  return edad >= 18;
}

function accesoCompleto(edad, esAdmin) {
  return puedeComprarAlcohol(edad) || esAdmin;
}

function ejecutarSi(condicion, accion) {
  condicion && accion(); // Ejecuta la función si la condición es true
}

// Ejemplo de secuencia de funciones
function validarUsuario(usuario) {
  return usuario.activo && usuario.verificado && !usuario.bloqueado;
}

Los operadores lógicos permiten definir flujos de decisión, ejecutar funciones condicionalmente y combinar múltiples condiciones en una sola expresión.

Tipos de memoria

  • Primitivos (string, number, boolean, null, undefined): Se almacenan en la stack.
  • Objetos y arrays: Se almacenan en la heap y la variable guarda una referencia.
