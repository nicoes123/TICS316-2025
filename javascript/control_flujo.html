Control de Flujo en JavaScript

En esta página aprenderás sobre las estructuras básicas de control de flujo en JavaScript: if, bucles (for, while, do...while), try/catch y switch case.

Condiciones: if

let edad = 20;
if (edad >= 18) {
  console.log("Eres mayor de edad");
} else {
  console.log("Eres menor de edad");
}

Bucles

Permiten repetir acciones varias veces:

Los bucles permiten ejecutar un bloque de código varias veces. Existen diferentes tipos:

Bucle for

Se usa para repetir un bloque un número determinado de veces.

for (let i = 0; i < 5; i++) {
            console.log(i);
          }

Bucle while

Repite el bloque mientras la condición sea verdadera.

let contador = 0;
          while (contador < 3) {
            console.log(contador);
            contador++;
          }

Bucle do...while

Ejecuta el bloque al menos una vez y luego repite mientras la condición sea verdadera.

let num = 0;
          do {
            console.log(num);
            num++;
          } while (num < 2);

Bucle for...of

Recorre los valores de un array o colección.

let frutas = ["manzana", "banana", "uva"];
          for (let fruta of frutas) {
            console.log(fruta);
          }

Bucle for...in

Recorre las propiedades de un objeto.

let persona = { nombre: "Ana", edad: 25, ciudad: "Santiago" };
          for (let clave in persona) {
            console.log(clave + ": " + persona[clave]);
          }

Break y Continue

break termina el bucle actual. continue salta a la siguiente iteración.

for (let i = 0; i < 5; i++) {
            if (i === 3) {
              break; // Termina el bucle cuando i es 3
            }
            console.log(i);
          }

          for (let i = 0; i < 5; i++) {
            if (i % 2 === 0) {
              continue; // Salta los números pares
            }
            console.log(i);
          }

Manejo de errores: try/catch

try {
  // Código que puede fallar
  let resultado = 10 / 0;
  console.log(resultado);
} catch (error) {
  console.log("Ocurrió un error:", error);
}

Switch case

let dia = "lunes";
switch (dia) {
  case "lunes":
    console.log("Inicio de semana");
    break;
  case "viernes":
    console.log("Fin de semana");
    break;
  default:
    console.log("Día normal");
}
